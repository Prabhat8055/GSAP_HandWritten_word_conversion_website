How it works

To convert your handwritten image to a computerized word using the handwritten website, follow these detailed steps. Begin by opening the image in Paint and access the image properties either by navigating to "File" in the top left corner or using the shortcut Ctrl + E. Once in image properties, set the units to pixels and adjust the image size to a width of 218 and a height of 70 pixels. Save the modified image to preserve the changes. Now, zoom in on your screen using Ctrl + mouse scroll up for a more detailed view. Select the pencil tool from the tools menu, and choose a size of 3 pixels. With the pencil tool, make any necessary adjustments or enhancements to the image. Save the image again after making these modifications. Next, visit the handwritten website and click on the "Convert" option. Navigate to the "Predict" page, where you'll find a designated area for uploading your image. Either drag and drop the modified image into this area or click to upload it. After uploading, a dialog box will appear, prompting you to select an empty folder. Ensure the chosen folder is empty, then click on "Upload Image." Following these steps, the website will process the uploaded image, and the computerized word will be displayed on the right side of the page. This detailed process ensures that your handwritten image is accurately converted into a computerized representation using the features provided by the website.
